Project Site Plan

Site Name: Bahins Car Sales

The reason I chose this project is because I plan to offer website and software services to local car dealers in our area, and this project gives me the opportunity to practice and improve my skills.

Site Purpose: The website will function as an online car store where users can browse and explore different car brands and models. Customers can view details such as year, price, and specifications, making it easy to find the right vehicle. The site will have a clean, user-friendly layout and will be responsive for both desktop and mobile devices, providing a seamless experience for all users.

SCENARIOS

Scenario 1: A user visits the homepage to browse featured cars and clicks the “Schedule a Test Drive” button to book a test drive.

Scenario 2: A user wants to see all available cars and clicks the “View Inventory” button to explore all models.
